# PWA Icons

This directory contains the icons required for PWA (Progressive Web App) functionality.

## Required Icons

- `icon-192x192.png` - 192x192 pixel icon for app installation
- `icon-512x512.png` - 512x512 pixel icon for high-resolution displays
- `maskable_icon.png` - Maskable icon for adaptive app icons on Android
- `profile-shortcut.png` - Icon for profile shortcut
- `marketplace-shortcut.png` - Icon for marketplace shortcut
- `badge.png` - Notification badge icon

## Screenshots (Optional)

- `home.png` - Screenshot of the home feed
- `marketplace.png` - Screenshot of the marketplace

## How to Generate Icons

You can generate these icons using various online tools or image editing software. Recommended tools:

1. [Figma](https://figma.com) - For designing and exporting icons
2. [Canva](https://canva.com) - For simple icon creation
3. [RealFaviconGenerator](https://realfavicongenerator.net/) - For generating complete PWA icon sets
4. [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator) - For generating all required assets from a single source image

## Icon Specifications

- Format: PNG
- Background: Transparent (for adaptive icons)
- Minimum size: 192x192 pixels for small icons
- Recommended size: 512x512 pixels for large icons
- Maskable icons should have extra padding for adaptive display
