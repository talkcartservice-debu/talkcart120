# Vetora Frontend

Vetora is a Web3 social commerce platform that combines social networking with marketplace functionality.

## Features

- Social feed with posts, comments, and reactions
- Marketplace for buying and selling products
- Real-time messaging system
- Video and image sharing
- Web3 integration with wallet connectivity
- Progressive Web App (PWA) support for mobile installation

## Progressive Web App (PWA)

Vetora supports installation as a Progressive Web App on mobile devices:

- Install the app on your home screen for quick access
- Native app-like experience with offline functionality
- Available on both Android and iOS devices

See [PWA Installation Guide](./PWA_INSTALLATION.md) for detailed instructions.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `npm run dev` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm start` - Runs the built app in production mode
- `npm test` - Runs tests

## Project Structure

- `/pages` - Next.js pages
- `/components` - React components
- `/hooks` - Custom React hooks
- `/contexts` - React context providers
- `/services` - API services and utilities
- `/lib` - Core library functions
- `/public` - Static assets
- `/styles` - CSS and styling files
- `/docs` - Documentation files
